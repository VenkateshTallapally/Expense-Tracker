# 💰 Expense Tracker

Expense Tracker is a simple Java console application for managing daily expenses. It allows users to add, view, search, update, and delete expense records. The project is built using Core Java, Object-Oriented Programming (OOP), Collections, and File Handling to provide an efficient and user-friendly expense management system.

---

## 🚀 Features

* Add new expenses
* View all expenses
* Search expenses by ID
* Update expense details
* Delete expenses
* Calculate total expenses
* Save data to a CSV file
* Load saved data automatically

---

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework (`ArrayList`)
* File Handling (CSV)
* Java LocalDate API

## ▶️ How to Run

1. Clone this repository.
2. Open the project in VS Code, IntelliJ IDEA, or Eclipse.
3. Compile the project.
4. Run `Main.java`.
5. Use the menu to manage your expenses.
